# Web Server Configuration Project

This project contains Bash scripts for configuring an Nginx web server on Ubuntu 16.04 LTS.

## Tasks

0. **0-transfer_file** - Transfer files to server using SCP
1. **1-install_nginx_web_server** - Install and configure Nginx
2. **2-setup_a_domain_name** - Domain name configuration
3. **3-redirection** - Configure 301 redirect
4. **4-not_found_page_404** - Custom 404 error page
5. **5-design_a_beautiful_404_page.html** - Creative 404 page design

## Requirements

- Ubuntu 16.04 LTS
- All scripts must be executable
- Pass Shellcheck (version 0.3.7)
- First line: `#!/usr/bin/env bash`
- Second line: Comment explaining the script
